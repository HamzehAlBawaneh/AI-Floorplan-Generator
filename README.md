An AI-powered procedural floorplan generation system that classifies land shapes using a ConvNeXt model and generates optimized building layouts based on user-defined constraints such as building type, budget, and land geometry.

## Overview

AI Floorplan Generator is an intelligent architectural design system that combines computer vision and procedural generation to automate the creation of building floorplans.

The system first classifies an uploaded land boundary image using a trained ConvNeXt deep learning model. Based on the detected land shape and user-defined constraints—including building type, budget, and design preferences—the backend generates an optimized floorplan. The generated layout is then presented through a modern React web interface.

## Features

- AI-based land shape classification using ConvNeXt
- Procedural floorplan generation
- Support for multiple building types
- User-defined constraints and preferences
- Budget-aware layout generation
- Geometry-aware room placement
- Interactive web interface
- Full-stack architecture with React, Node.js, and Python
